# 供应商管理 - 菜单配置说明

## 在若依后台配置菜单

1. 登录若依后台管理系统
2. 进入：系统管理 → 菜单管理
3. 点击"新增"按钮，添加以下菜单：

### 父级菜单
- **菜单名称**：供应商管理
- **路由地址**：supplierManage
- **组件路径**：Layout
- **显示排序**：根据需要设置
- **菜单类型**：目录
- **菜单状态**：显示
- **图标**：shopping (或自定义)

### 子菜单
- **上级菜单**：供应商管理
- **菜单名称**：供应商列表
- **路由地址**：supplierList
- **组件路径**：supplierManage/supplierList/index
- **权限标识**：supplierManage:supplierList:list
- **显示排序**：1
- **菜单类型**：菜单
- **菜单状态**：显示

## 前端路由配置

✅ 已在 `src/router/index.js` 中添加路由配置

## 企业别名搜索框说明

企业别名搜索框已在以下位置实现：
- 文件：`src/views/supplierManage/supplierList/components/searchSection.vue`
- 位置：第18-27行
- 字段名：`nickName`
- 功能：支持精确搜索和模糊搜索，可与其他条件组合使用
